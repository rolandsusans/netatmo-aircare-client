# Netatmo-aircare-client
Client to extract data from: https://dev.netatmo.com/apidocumentation/aircare device.

Measurement information:
 - Temperature
 - Humidity
 - CO2
 - Noise
 - Pressure
 - AbsolutePressure
 - LastMeasure UTC time of last measurement
 - HealthIdx

Device information:
 - WifiStatus
